# INCALIN-UNSAM - Unofficial PhD Thesis Template (v1.0 - 2022)
(below there is an English version)

Este template fue producido modificando el original de TU Delft (v1.5) citado más abajo.

  <img align="centre"  src="https://github.com/realmariano/PhDthesis/blob/c8f9b34edc5079d594f54d3de8ab412e8ede29f3/figures/template.png" alt="Preview of the Template" width="85%" />

En el archivo /layout/.unsam-incalin-phd_v2.cls, encontrará más información acerca de cómo modificar este template a sus necesidades. También, el original utilizaba una fuente distinta, como pude verse más abajo. Es posible cambiarla nuevamente simplemente cambiando los parámetros indicado en el archivo de la clase.
Todo lo demás es autoexplicativo, basta mirar el main.tex y archivos en /layout/ para comenzar a modificar y utilizar este template.

La estructura de archivo s fue pensada para mantener todo manejable y sencillo. 

Si no usa un sistema de manejo de referencias, tal vez le sirva mirar  https://github.com/realmariano/wiki_labo.

__English version__

This template was produce by modifying the one from TU Delft (v1.5) cited below.

Please see /layout/.unsam-incalin-phd_v2.cls, there you will find extra info in how to modify this template to your needs. Also the original template used a different font as you can see below. It is possible to change this easely in the class file.

Eveything else is self explanatory in the notes included in the files, mostly by looking main.tex and the files in /layout/. 
The tree is thought to keep things clean and manageable.

If you do not use some kind of reference management system, perhaps you will find https://github.com/realmariano/wiki_labo helpful.

This template works with _pdfLaTeX_, _XeLaTeX_ and _LuaLaTeX_, as it supports TrueType and OpenType fonts. 

_BibLaTeX_ is used for the bibliography, backend _biber_.
